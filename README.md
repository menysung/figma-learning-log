## 공부한 거 요약
1. UI의 요소 이해하기 & 와이어프레임 실습
   - 메인 탭 (main): 아이콘 플러그인을 활용해 상단 바, 검색, 하단 레이아웃 틀 잡기
   - 상세 탭 (detail): 아코디언 메뉴 구조 설계 및 리스트 배치
   - 검색 탭 (search): 최근 검색어 리스트 및 인기 카테고리 원형 UI 배치
2. 오토 레이아웃(Auto Layout) 맛보기
   - 요소가 늘어나거나 줄어들 때 반응형으로 텍스트와 레이아웃이 유동적으로 변하게 만드는 핵심 기능 맛보기
- 오토 레이아웃 생성: Shift + A
- 상위 부모 프레임 선택: Shift + Enter
- 하위 자식 레이어 다이렉트 선택: Ctrl + 마우스 클릭 (맥: Cmd + 클릭)
- 오토 레이아웃 해제 (프레임 깨기): Ctrl + Shift + G

## 노트 
- 아코디언 메뉴처럼 컴포넌트가 아래로 늘어나는 UI를 만들 때는 단순 '그룹(Group)'이 아니라 상위와 하위 요소를 계층 구조에 맞춰 '오토 레이아웃(Shift + A)'으로 묶어주어야 겹치지 않고 자연스럽게 아래로 밀려 내려간다.
- 가로 정렬 시 글자가 겹치거나 오른쪽 끝으로 가지 않는다면, 부모 프레임의 가로폭 설정을 `Hug`에서 `Fill container`로 변경하고 간격을 `Auto` (Space between)로 찢어주면 해결됨!
